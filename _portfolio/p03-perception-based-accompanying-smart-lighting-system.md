**Person-In-Charge, Embedded Engineer**

## Overview
This is an engineering project aiming at improving the energy efficiency and intelligence of the current lighting systems which are deployed in rural areas. Our system mainly consists of several parts: Dynamic & Linear Lighting, Vision-and-Sensors-based Smart Perception & Decision, Edge-Cloud Control, Long-Range Communications etc.

## Responsible for:
- Embedded systems and software, and integration of different parts
- Long-Range communications network built via LoRa
- Cloud Service Platform development, including backend and frontend
